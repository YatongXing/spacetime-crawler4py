CompSci 260P Submissions

Homework Assignments

  • Homework submissions are to be uploaded to the class gradescope account before 10:00am of the day listed on the Class Schedule on which their solutions are discussed
    • No late submissions will be accepted
    • Instructions for using gradescope can be found at https://www.gradescope.com/help#help-center-section-student-workflow
  • One problem from each homework problem set will be graded for correctness and will score up to a total of 5 points,
    while up to 5 additional points will be awarded for showing work on the other problems in that set
  • Collaboration on homework, or asking for or receiving assistance with homework problems, is not permitted
  • Solutions for homework problems will be reviewed in lecture

Projects

  • Project submissions are to be uploaded to the class gradescope account before 10:00am of the submission deadline date listed on the Class Schedule
    • Each project requires separate submissions of one source code file and one analysis file (.pdf or .txt)
    • No late submissions will be accepted
    • Instructions for using gradescope can be found at https://www.gradescope.com/help#help-center-section-student-workflow
  • Code must be original, and may not be copied or shared from any other source, except as provided by the class instructor

Midterms

  • Midterm examinations are conducted in class at the regularly scheduled time
  • Midterm examinations are closed-book, closed-note
  • Collaboration on examinations, or asking for or receiving assistance with examination problems, is not permitted
  • Solutions for examination problems will be reviewed in lecture
