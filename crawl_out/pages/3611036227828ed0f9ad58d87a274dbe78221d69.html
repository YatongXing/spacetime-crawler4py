Patient Coding

patient billing codes

MRN

Medical record number which is a unique identifier for an individual patient.

SOURCE_KEY, SOURCE_NAME, REF_BILL_CODE_SET_NAME

SOURCE_KEY is an integer representing the reference billing code set being used, SOURCE_NAME is the name of the reference billing code set being used, and REF_BILL_CC is the abbreviation for the code set being used.
 

NAME, REF_BILL_CODE

NAME is the name of what the patient is being billed for and the REF_BILL_CODE is the corresponding billing code.
