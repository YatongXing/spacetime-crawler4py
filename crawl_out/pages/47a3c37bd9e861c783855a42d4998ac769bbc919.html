Protocol of Using Slack

  • Add a "real picture" of you to your profile so that people can "see" you.
  • Channel "#Announcements": for the class instructor and TA to send announcements to the class. No replies from students.
  • Channel "#General": for the class to have discussions about general topics.
  • Channel "#Projects": for the class to have discussions about all the projects.
  • Channel "#Random": for the class to have discussions about random topics, including those not directly related to the class.
  • If your questions are of interest to the class and not private, you must send them in one of the public channels.
  • For your private questions, create a direct chat with the TA, Zuozhi. Include the instructor only if needed.
  • The TA and the instructor only check Slack messages periodically. So please don't expect immediate responses.
